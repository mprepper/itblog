ЗАБОНҲОИ МАШҲУРИ БАРНОМАСОЗӢ


C - як забони барномасозии ба таври статикӣ тартибдодашудаи таъиноти умумӣ мебошад. Дар байни солҳои 1969 ва 1973 аз ҷониби Деннис Ритчи таҳия шудааст. Дар ибтидо барои татбиқи системаи оператсионии UNIX таҳия карда шудааст.

C++ - забони барномасозии таъиноти умумии компилятсияшаванда, ки ба таври статикӣ тартиб дода шудааст.Барномасозии мурофиавӣ, барномасозии ба объект нигаронидашуда, барномасозии умумиро дастгирӣ мекунад.

Javascript - забони барномасозии мултипарадигмавӣ.
Услубҳои ба объект нигаронидашуда, императивӣ ва функсионалиро дастгирӣ мекунад.
Татбиқи мушаххасоти ECMAScript мебошад.

Java - як забони пурқувват тартибдодашуда, ба объект нигаронидашуда, барои мақсадҳои умумӣ.
Ин як забони барномасозии зуд, бехатар ва боэътимод барои ҳама чиз аст. Аз барномаҳои мобилӣ то технологияҳои сервер.

Python - забони
барномасозии мултипарадигмавии
сатҳи баланд бо чопкунии пурқуввати динамикӣ,
ки ба баланд бардоштани ҳосилнокии барномасозон нигаронида шудааст.

Swift - забони барномасозии кушода, компилятсияшаванда, бисёрпарадигмавӣ
ва умумӣ, ки аз ҷониби
Apple таҳия ва нигоҳдорӣ мешавад.
Дар таҳияи барномаҳо барои macOS, iOS, iPadOS истифода мешавад.

C# - забони барномасозии ба объект нигаронидашудаи таъиноти умумӣ. Дар солҳои 1998-2001 аз ҷониби як гурӯҳи муҳандисони Microsoft
ҳамчун забони таҳияи барномаҳо барои .NET Framework ва .NET Core таҳия шудааст.

Assembly - дар муқоиса бо забонҳои сатҳи баланд, ки ба татбиқи мушаххаси системаи компютерӣ алоқаманд нестанд, забони барномасозии сатҳи паст ҳисобида мешавад. Сохтори асосии забон мнемоника ё мнемокод мебошад.

Go - як забони барномасозии бисёр риштаи компилятсияшаванда, ки аз ҷониби Google таҳия шудааст. Услубҳои ба объект нигаронидашуда ва функсионалиро дастгирӣ мекунад. Он расман дар соли 2009 муаррифӣ карда шуд.

PHP - забони скриптӣ, ки барои таҳияи барномаҳои веб истифода мешавад. Дар айни замон аз ҷониби аксари провайдерҳои хостинг дастгирӣ карда мешавад, он яке аз забонҳои пешқадамест, ки барои сохтани вебсайтҳои динамикӣ истифода мешавад.

HTML - (HyperText Markup Language — «забони аломатгузории гиперматн») — забони стандартизатсияшудаи гиперматн барои дидани саҳифаҳои веб дар браузер. Онро олими бритониёӣ
Тим Бернерс-Ли таҳия кардааст.

CSS - (Cascading Style Sheets «варақаҳои услубӣ») — забони расмӣ барои рамзкушоӣ ва тавсифи намуди ҳуҷҷате, ки бо истифода аз забони аломатгузорӣ навишта шудааст. Инчунин метавонад ба ҳама гуна ҳуҷҷатҳои XML татбиқ карда шавад.
